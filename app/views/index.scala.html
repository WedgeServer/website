lock Automatic TLS

Wedge automatically integrates with Let's Encrypt to provide painless HTTPS support out of the box.

No need for CSR s, email verification or manual approval. Wedge uses Lego , a pure Go ACME client that can automatically prove domain ownership and request free SSL certificates.

And there's no need to ever manually renew another certificate. Everything is taken care of for you.

apps Plugin Support

Wedge has support for a wide array of plugins. Wedge is based on the Caddy source code and maintains API compatibility so all Caddy plugins should be fully compatible.

If you're unfamiliar with Caddy, that means support for rate limiting, DNS providers like Namecheap™ for domain ownership verification, caching and more. Check out the full plugins list for more.
